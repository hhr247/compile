### 使用方法

运行source.exe程序

输入将要进行分析的文件名。

程序会输出当前栈，并且在最后输出是否成功，若是成功，会输出成功，否则输出失败原因。

目前以实现功能：

- 多表达式，使用分号分隔
- 赋值表达式
- 二元算术表达式
- 一元表达式
