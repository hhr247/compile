# readme

#### 源文件

源文件为source.cpp

编译出来的程序为source.exe

#### 使用方法

此时并未完成预处理的工作，如#等内容并未完成，需要对输入文件进行处理，将所有预编译的内容清除。

程序的输入为文件名 如: demo.c

程序的输出为tokens.txt，其中包含对单词的分析以及代号的转换，所处的行数。

#### 程序功能

程序能够识别C语言的输入文件。

该词法分析器能够实现字符与字符串的识别，注释的识别，多进制实数与小数的识别，标识符和关键字，界符和运算符的识别，基本包含了大部分的C所支持的内容。
